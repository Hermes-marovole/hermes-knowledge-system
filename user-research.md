## 1. 研究概述

### 1.1 背景
本研究报告服务于一家初创型合成生物学公司，主营AKK益生菌、镁等营养保健品，在美国和中国两地销售。需要开发AI产品覆盖：
- 面向中国用户的小程序
- 面向美国用户的iOS/安卓App
- 面向专业营养师的营养学助手WebApp

### 1.2 研究目标
1. 研究中国营养保健品消费者画像（年龄、健康诉求、购买决策因素）
2. 研究美国supplement消费者画像（与中国市场的差异）
3. 分析专业营养师的工作场景和AI助手需求
4. 输出用户场景文档，包含：用户画像卡片、使用场景故事、痛点与AI解决方案映射

### 1.3 数据来源

### 2.2 中国市场关键数据洞察

- **市场规模**：2024年中国营养健康食品市场规模达2,602亿元，年复合增长率6.3%
- **消费主力**：31-40岁人群是消费主力军，但银发人群渗透率持续攀升
- **渠道分布**：电商渠道占比48.02%（2023年），淘宝/天猫平台市占率42%
- **功效偏好**：口服美容占比最高，体重管理、运动营养、女性健康、肠道健康增长强劲
- **进口趋势**：进口品牌份额从2019年12.2%提升至2023年19.3%，美国、澳大利亚、德国为主要来源国

### 3.2 美国市场关键数据洞察

- **市场规模**：2024年美国维生素矿物质补剂(VMS)市场605亿美元，全球第一
- **世代差异**：
  - Baby Boomers：34%购买矿物质，品牌忠诚度高（39%重复购买成功产品）
  - Gen X：22%购买益生菌，注重性价比（46%认为重要）
  - Millennials：23%购买益生菌，接受度最高
  - Gen Z：31%未购买过补剂，但镁补充剂用户增长最快
- **趋势特征**：
  - 消费者愿意为"临床验证"标签多付20%以上
  - 个性化营养解决方案快速增长
  - 镁补充剂成为2024年增长最快的品类（改善血压、睡眠、减压）
- **决策因素排序**：健康目标 > 品牌认知 > 医疗提供者推荐 > 便利性 > 价格 > 可持续与伦理实践

## 6. 典型使用场景故事

### 场景1：中国中产妈妈为全家选择益生菌产品

**用户**：李女士，35岁，上海外企职员，家有6岁儿童和65岁父母

**场景**：换季时节，李女士担心家人免疫力不足。她在微信小程序中打开AI营养助手，首先为自己进行健康评估：AI通过问卷了解她久坐办公、经常熬夜、饮食外卖为主的现状，结合她上传的体检报告，识别出肠道菌群失衡和轻度代谢异常的风险。AI推荐AKK益生菌产品，解释其"通过激活GLP-1通路调节代谢"的科学原理，并为她制定个性化服用方案。

随后，李女士为便秘困扰的父亲和挑食的孩子分别进行儿童/老年模式评估。AI根据年龄差异调整问卷，推荐适合儿童的益生菌剂型和适合老年人的免疫调节方案，生成家庭营养档案。李女士一键下单三套产品，AI设置服用提醒，并约定2周后回访收集反馈。

**痛点解决**：
- 产品选择困难 → AI基于健康数据精准推荐
- 家庭成员需求差异 → 多档案管理+年龄适配
- 服用依从性低 → 智能提醒+效果追踪

### 场景2：美国千禧一代运动员优化运动表现

**用户**：Mike，28岁，加州软件工程师，业余马拉松爱好者

**场景**：Mike希望提升马拉松训练效果，在iOS App中完成注册后，AI引导他连接Apple Watch和Strava账户，自动同步运动数据。Mike填写饮食习惯问卷，说明自己尝试素食但担心蛋白质不足。

AI分析其运动强度数据（每周跑量50km+）、心率变异性、睡眠质量，识别出镁缺乏风险（肌肉痉挛、恢复缓慢），结合素食饮食特点推荐镁补充剂+植物蛋白组合。AI生成训练期营养方案，包括训练前后营养时机、补水电解质策略。

Mike发现AI推荐的补剂剂量与训练周期联动：高强度训练周自动增加镁和抗氧化剂建议， tapering期调整建议。他通过App直接购买产品，AI将服用提醒同步到Apple Watch，并每周生成训练-营养关联分析报告。

**痛点解决**：
- 素食运动员营养缺口 → 精准营养素识别
- 运动表现与营养关联不清 → 数据整合分析
- 补剂剂量与训练周期不匹配 → 动态方案调整

### 场景3：营养师使用AI助手提升客户管理效率

**用户**：王营养师，某三甲医院临床营养科

**场景**：王营养师每天面临大量糖尿病患者的营养咨询。使用AI营养师WebApp后，工作流程发生质变：

晨间，系统自动整理今日预约患者数据：从医院HIS系统同步最新血糖检测数据，AI预分析患者近两周血糖波动模式，标记出需要重点关注的三位患者。

面诊时，王营养师快速浏览AI生成的患者摘要：包括饮食记录分析（患者上传的饮食照片经AI识别后生成的营养摄入估算）、血糖-饮食相关性分析、药物-营养相互作用提醒。AI已预生成三套膳食方案初稿，王营养师根据患者实际口味偏好做微调即可。

患者回家后，AI接手日常随访：通过微信小程序推送个性化饮食提醒，患者上传的饮食照片继续被分析，偏离方案时自动提示调整。两周后，AI自动生成效果评估报告，血糖控制达标率、依从性评分一目了然。

**痛点解决**：
- 数据采集耗时 → 多源数据自动整合
- 方案制定重复劳动 → AI初稿+人工精调
- 离院后患者失联 → 自动化随访闭环
- 服务人数受限 → 效率提升服务更多患者

### 场景4：Z世代消费者科学入门营养补剂

**用户**：小林，22岁，大四学生，经常熬夜刷题、饮食不规律

**场景**：小林在小红书看到AKK益生菌的种草笔记，但对功效存疑。他打开品牌小程序中的AI问答功能，用自然语言提问："这个益生菌和普通的有什么区别？"

AI没有直接推销，而是先通过5个快速问题了解他的作息、饮食、肠道症状。基于"熬夜+外卖+轻度便秘"的画像，AI用通俗语言解释肠道菌群失衡的原理，说明AKK菌通过Amuc_1100蛋白修复肠黏膜屏障的科学依据，展示相关临床研究数据。

小林继续问："我预算有限，这个产品贵吗？"AI推荐适合学生党的入门装规格，并建议配合饮食调整（减少夜间进食、增加膳食纤维）以提升效果。AI生成"学生党肠道健康方案"，包含补剂+饮食+作息建议，设置成本仅为每日X元。

被AI的专业性和不推销态度打动，小林下单试用装。AI设置14天体验跟踪计划，每天推送简单的肠道健康小贴士，第7天和第14天自动发送反馈问卷。

**痛点解决**：
- 信息可信度存疑 → 循证医学+通俗解释
- 预算约束 → 灵活规格推荐+综合方案
- 缺乏持续动力 → 轻量化跟踪+教育内容

### 场景5：美国中年用户整合多源健康数据获得个性化方案

**用户**：Jennifer，52岁，德州会计师，有家族糖尿病史

**场景**：Jennifer关注代谢健康，在App中开启综合健康评估。她授权连接：
- Oura Ring的睡眠数据
- 年度体检报告（上传PDF经AI解析）
- 23andMe基因检测数据（已上传的APOE和糖尿病风险基因型）

AI整合多维度数据，识别出她的核心风险：胰岛素抵抗早期迹象、睡眠碎片化影响代谢、家族风险叠加。AI没有简单推荐通用方案，而是：

1. 解释风险关联："您的睡眠效率仅72%，深度睡眠不足会影响胰岛素敏感性，这与您的家族史形成叠加风险"
2. 分层干预建议：一级（优先）- 镁补充剂改善睡眠；二级 - AKK益生菌调节代谢；三级 - 饮食时间调整
3. 与医生协作：AI生成总结报告，Jennifer可以一键分享给她的家庭医生

AI根据Jennifer的反馈动态调整：第一月重点追踪睡眠改善效果，Oura数据自动同步，睡眠效率提升至80%后，启动第二阶段代谢指标关注。

**痛点解决**：
- 数据孤岛问题 → 多设备/多来源整合
- 复杂健康信息难以理解 → 风险关联可视化
- 医生沟通障碍 → 结构化报告一键分享
- 长期依从性 → 分阶段目标+动态激励
