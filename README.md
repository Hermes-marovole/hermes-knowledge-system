# Hermes Knowledge System

三层架构知识管理系统，用于组织和管理 AI 领域专家技能与知识资源。

## 🏗️ 三层架构

```
hermes-knowledge-system/
├── karpathy-skills/    # Karpathy 专家技能层
├── hermes-skills/      # Hermes 技能层
└── llm-wiki/           # LLM-Wiki 知识库层
```

### 1. Karpathy Skills (`/karpathy-skills/`)

存放来自 Andrej Karpathy 及其风格的专家级技能文档，包括：
- 深度学习与神经网络核心技术
- LLM 训练与微调方法
- 高效算法与系统优化技巧
- 前沿 AI 研究解读与实践

### 2. Hermes Skills (`/hermes-skills/`)

Hermes 智能体系统的专属技能集合：
- 工具调用与编排模式
- 任务分解与规划策略
- 多智能体协作协议
- 环境感知与自适应机制

### 3. LLM-Wiki (`/llm-wiki/`)

大型语言模型相关知识百科：
- 模型架构与技术原理
- 训练数据与评估基准
- 应用场景与最佳实践
- 行业动态与趋势分析

## 📚 使用指南

1. **Karpathy Skills**: 参考专家级技能解决复杂技术问题
2. **Hermes Skills**: 查阅智能体系统设计与实现方案
3. **LLM-Wiki**: 获取 LLM 领域基础概念与最新资讯
