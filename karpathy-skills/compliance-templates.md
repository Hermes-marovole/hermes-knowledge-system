## 一、中国法规允许/禁止表述对照表

### 1.1 产品类型定义

| 产品类型 | 法规定义 | 标识要求 | 声称限制 |
|---------|---------|---------|---------|
| **普通食品** | 供人食用或饮用的成品和原料 | 无特殊标识 | 仅限营养成分功能声称(GB 28050) |
| **保健食品** | 声称具有特定保健功能或补充营养素 | "蓝帽子" + 批准文号 | 24项保健功能目录(2023版) |
| **新食品原料** | 无传统食用习惯的食品原料 | 标注"不适宜人群" | 不得声称功效 |
| **特殊医学用途配方食品** | 满足特殊医学状态营养需求 | 特定标识 | 需临床验证 |

### 1.2 普通食品 - 禁止用语清单

#### ❌ 绝对禁止表述

| 类别 | 禁止用语示例 | 违规后果 |
|------|-------------|---------|
| **疾病治疗** | 治疗、治愈、康复、疗效、疗程、医药级、药方 | 违反《广告法》第17条 |
| **保健功能** | 保健功能、调节、增强、改善(功能性描述) | 违反《食品安全法》第73条 |
| **疾病暗示** | 降三高、控糖、降脂、降压、抗肿瘤 | 可能被视为疾病声称 |
| **夸大宣传** | 神奇、特效、神效、100%有效、无副作用 | 虚假宣传 |
| **医疗用语** | 处方、药用、疗程、临床验证(非真实)、医生建议 | 易与药品混淆 |

#### ⚠️ 高风险灰色地带

| 风险等级 | 表述 | 判定依据 | 建议替代话术 |
|---------|------|---------|-------------|
| 🔴 高风险 | "祛湿"、"降火"、"排毒" | 中医术语，可能被认定为保健功能 | "传统饮食文化概念，非功效声称" |
| 🔴 高风险 | "逆龄"、"冻龄"、"抗衰" | 被认定为暗示保健功能 | "日常养护"、"支持健康生活方式" |
| 🟡 中风险 | "辅助减重" | 接近保健功能声称 | "配合健康饮食和运动" |
| 🟡 中风险 | "调节肠道" | 保健食品功能目录用语 | "含有膳食纤维/益生菌" |

### 1.3 保健食品 - 允许声称(24项功能目录)

| 序号 | 允许声称功能 | 适用产品举例 | 注意事项 |
|------|-------------|-------------|---------|
| 1 | 有助于增强免疫力 | 含人参、灵芝产品 | 需标注"本品不能替代药物" |
| 2 | 有助于抗氧化 | 含维生素C、E产品 | 标注抗氧化成分含量 |
| 3 | 辅助改善记忆 | DHA、磷脂酰丝氨酸 | 需有相应人体试食试验 |
| 4 | 缓解视觉疲劳 | 叶黄素、蓝莓提取物 | 标注适用人群 |
| 5 | 清咽润喉 | 金银花、胖大海 | 标注不适宜人群 |
| 6 | 有助于改善睡眠 | 褪黑素、酸枣仁 | 标注每日最大用量 |
| 7 | 缓解体力疲劳 | 人参、红景天 | 不涉及提高运动表现 |
| 8 | 耐缺氧 | 红景天、角鲨烯 | 高原特定用途 |
| 9 | 有助于控制体内脂肪 | 膳食纤维、茶多酚 | 需配合运动饮食 |
| 10 | 有助于改善骨密度 | 钙、维生素D | 标注钙含量 |
| 11 | 改善缺铁性贫血 | 铁制剂 | 标注铁含量和适宜人群 |
| 12-16 | 改善痤疮/黄褐斑/皮肤水份 | 胶原蛋白、维生素 | 不得涉及医疗美容 |
| 17 | 有助于调节肠道菌群 | 益生菌 | 标注活菌数 |
| 18 | 有助于消化 | 消化酶、山楂 | 区分保健食品与普通食品 |
| 19 | 有助于润肠通便 | 膳食纤维 | 标注膳食纤维含量 |
| 20 | 辅助保护胃粘膜 | 蜂胶、砂仁 | 不涉及治疗胃病 |
| 21-23 | 维持血脂/血糖/血压健康水平 | 红曲、苦瓜提取物 | 强调"维持健康水平"而非"治疗" |
| 24 | 对化学性肝损伤有辅助保护作用 | 水飞蓟、葛根 | 不涉及治疗肝病 |

### 1.4 营养成分功能声称(GB 28050标准)

#### ✅ 普通食品可使用的标准话术

| 营养素 | 允许声称 | 含量要求 |
|--------|---------|---------|
| 镁 | "镁是能量代谢、组织形成和骨骼发育的重要物质" | ≥30% NRV/100g |
| 钙 | "钙有助于骨骼和牙齿更坚固" | ≥30% NRV/100g |
| 维生素C | "维生素C有助于维持皮肤和黏膜健康" | ≥30% NRV/100g |
| 膳食纤维 | "膳食纤维有助于维持正常的肠道功能" | ≥3g/100g或1.5g/100mL |
| 蛋白质 | "蛋白质是人体的主要构成物质并提供多种氨基酸" | ≥12% NRV/100g |
| 益生菌 | "含有活性益生菌" | 标注菌种和活菌数 |

> ⚠️ 必须使用GB 28050规定的标准用语，**不得自行扩展或修改**

## 二、美国FDA结构/功能声明模板

### 2.1 膳食补充剂声称类型

| 声称类型 | 定义 | 证据要求 | 示例 |
|---------|------|---------|------|
| **Structure/Function Claim** | 描述营养素对正常结构或功能的影响 | 充足依据 | "钙支持骨骼健康" |
| **Health Claim** | 描述与疾病风险的关联 | FDA授权或合格健康声称 | "钙可能降低骨质疏松风险" |
| **Nutrient Content Claim** | 描述营养素含量水平 | 符合定义标准 | "高钙"、"钙的良好来源" |

### 2.2 Structure/Function Claim 合规要点

#### 必须遵守的规则 (21 CFR 101.93)

```
✅ 允许:
- "镁支持正常的肌肉功能"
- "益生菌支持消化健康"
- "本产品支持免疫健康"

❌ 禁止:
- "镁治疗肌肉痉挛" (疾病声称)
- "益生菌预防结肠癌" (疾病声称)
- "本产品治愈感冒" (疾病声称)
```

#### 必须包含的免责声明

```
"This statement has not been evaluated by the Food and Drug Administration. 
This product is not intended to diagnose, treat, cure, or prevent any disease."

(本声明未经食品药品监督管理局评估。本产品不用于诊断、治疗、治愈或预防任何疾病。)
```

#### FDA审查重点

| 审查要素 | 合规要求 |
|---------|---------|
| 证据依据 | 上市前30天通知FDA，需有充足依据 |
| 声称措辞 | 不得涉及疾病诊断、治疗、治愈、预防 |
| 标签标识 | 必须包含上述免责声明 |
| 成分安全 | 遵循NDI(新膳食成分)通知要求 |

### 2.3 FDA推荐话术模板

#### ✅ 结构/功能声称范例

| 产品类别 | 合规话术 | 不合规话术 |
|---------|---------|-----------|
| **益生菌** | "支持肠道微生物平衡"、"促进消化健康" | "治疗肠易激综合征"、"预防结肠癌" |
| **镁补充剂** | "支持肌肉和神经功能"、"促进放松" | "治疗心律失常"、"治愈焦虑症" |
| **益生元** | "支持益生菌生长"、"促进肠道健康" | "治疗便秘"、"治愈结肠炎" |
| **蛋白质** | "支持肌肉合成"、"帮助维持瘦体重" | "增肌"、"治疗营养不良" |

#### ⚠️ 疾病 vs 结构/功能 边界指南

| 疾病声称(禁止) | 结构/功能声称(允许) |
|---------------|-------------------|
| lowers cholesterol (降低胆固醇) | helps maintain cholesterol levels already in the normal range (帮助维持已在正常范围的胆固醇水平) |
| prevents osteoporosis (预防骨质疏松) | helps support bone health (帮助支持骨骼健康) |
| treats insomnia (治疗失眠) | promotes relaxation and restful sleep (促进放松和安稳睡眠) |
| reduces risk of heart disease (降低心脏病风险) | supports cardiovascular health (支持心血管健康) |
| cures digestive disorders (治愈消化障碍) | supports healthy digestion (支持健康消化) |

## 三、敏感话题保守措辞指南

### 3.1 减重/体重管理话题

#### 中国合规策略

| 敏感词 | 替代话术 | 合规依据 |
|-------|---------|---------|
| 减肥 | "配合健康饮食和运动的生活方式" | 避免疾病/功能声称 |
| 燃脂 | "能量代谢支持" | 描述机制而非结果 |
| 瘦身 | "体型管理" | 非医学术语 |
| 控制体重 | "有助于控制体内脂肪"(仅限保健食品) | 保健食品功能目录第9项 |

#### 推荐话术模板
```
保守版本:
"本产品含有膳食纤维/蛋白质，可作为健康饮食的一部分。
体重管理需要综合考虑饮食、运动和生活方式因素。"

循证版本:
"根据XX研究，在控制饮食和增加运动的基础上，
补充XX成分与体重变化相关。个体差异可能存在。"
```

### 3.2 血糖话题

#### 中国合规策略

| 敏感词 | 替代话术 | 合规依据 |
|-------|---------|---------|
| 降血糖 | "有助于维持血糖健康水平"(保健食品) | 功能目录第22项 |
| 控糖 | "糖代谢支持"、"餐后管理" | 描述机制 |
| 治疗糖尿病 | 绝对禁止 | 医疗声称 |
| 替代胰岛素 | 绝对禁止 | 严重违规 |

#### 推荐话术模板
```
合规表述:
"本产品含有铬/膳食纤维，铬是维持正常血糖代谢所需的微量元素。
本品不能替代药物治疗，糖尿病患者使用前应咨询医生。"

风险警示:
"任何宣称能'根治'糖尿病、'替代'胰岛素的说法均属违法虚假宣传。
血糖管理应在医生指导下进行。"
```

### 3.4 肠道健康话题

| 产品类型 | 保守话术 | 激进话术(禁止) |
|---------|---------|---------------|
| 益生菌 | "含有活性益生菌"、"支持肠道菌群平衡" | "治疗肠炎"、"根治便秘" |

## 四、跨境销售合规要点

### 4.1 跨境电商模式

| 场景 | 境内合规要求 | 境外主体责任 |
|-----|-------------|-------------|
| **保税区模式** | 境内代运营需遵守中国法规 | 产品质量符合原产国标准 |
| **直邮模式** | 境内广告推广需合规 | 境外网站不受直接监管 |
| **直播带货** | 主播和MCN需审核话术 | 品牌方提供合规资料 |

### 4.2 典型案例警示

**案例1**: 京朝市监处罚〔2024〕109号
- **案情**: 主播推广日本植物酵素，使用"抑制糖分吸收、吸附油脂"
- **处罚**: 罚款6万元
- **启示**: 境内主体参与跨境产品推广，需严格遵守中国法规

**案例2**: 益生菌直播违规
- **常见违规**: 宣称"提高免疫力"(保健功能)的普通食品益生菌
- **处罚依据**: 《广告法》第17条、《食品安全法》第73条

### 4.3 跨境合规建议

```
境内推广必查清单:
✓ 产品是否为普通食品/保健食品/新食品原料?
✓ 是否取得相应批准文号?
✓ 话术是否涉及疾病治疗/保健功能?
✓ 是否包含合规免责声明?
✓ 是否标注"本产品为境外产品，符合原产国标准"?
```

## 五、AI营养师话术流程模板

### 5.1 标准对话流程

```
第一步: 需求确认
"请问您想了解哪方面的营养支持? 是关于睡眠、消化还是其他方面?"

第二步: 信息收集(如适用)
"您目前的饮食和生活习惯是怎样的? 有没有正在服用的药物?"

第三步: 知识分享
"根据现有研究，XX营养素可能支持XX功能。例如，一项发表在XX的研究显示..."

第四步: 产品建议(谨慎)
"如果您考虑补充，XX形式可能有较好的吸收率。但请注意..."

第五步: 风险提示
"重要提示: 任何补充剂都不能替代药物治疗。如果您有XX情况，建议先咨询医生。"
```

### 5.2 高风险话题应对

#### 当用户询问"能不能治病"
```
标准回应:
"营养补充剂不能替代药物治疗疾病。如果您有健康问题，
建议及时就医。营养支持是健康管理的辅助手段，而非治疗手段。
我主要可以帮您了解不同营养素的日常需求和食物来源。"
```

#### 当用户询问"快速减肥"
```
标准回应:
"健康的体重管理需要循序渐进。根据世界卫生组织的建议，
每周减重0.5-1公斤是较为安全和可持续的速度。
极端的减重方法可能带来健康风险。
我可以为您介绍一些有助于健康饮食的营养素知识。"
```

#### 当用户询问"降血糖/降压"
```
标准回应:
"血糖/血压管理是医疗问题，应在医生指导下进行。
某些营养素可能在整体健康中发挥作用，但绝对不能
替代药物治疗。如果您正在接受相关治疗，
请务必告知医生您考虑使用的任何补充剂，以避免相互作用。"
```

## 六、合规自查清单

### 6.1 内容发布前检查

| 检查项 | 是/否 | 备注 |
|-------|-------|------|
| 是否涉及疾病名称? | ☐ | 如糖尿病、高血压、癌症等 |
| 是否使用治疗性动词? | ☐ | 治疗、治愈、康复、疗效等 |
| 是否声称保健功能? | ☐ | 24项功能目录外 |
| 是否有夸大表述? | ☐ | 神奇、100%有效、无副作用等 |
| 是否标注免责声明? | ☐ | 保健食品必须 |
| 是否注明适用人群? | ☐ | 特别是孕妇、儿童、患者 |
| 是否引用虚假研究? | ☐ | 确保来源可靠 |

### 6.2 违规应对流程

```
发现潜在违规 → 立即暂停相关内容 → 评估风险等级 →
修改/撤下内容 → 记录备案 → 培训改进 → 定期复查
```
